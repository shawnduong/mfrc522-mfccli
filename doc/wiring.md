# Wiring

An RFID-RC522 reader/writer and an Arduino Uno were used for testing. Stability is uncertain on other devices.

| Reader/Writer | Arduino    |
|---------------|------------|
| SDA           | 10         |
| SCK           | 13         |
| MOSI          | 11         |
| MISO          | 12         |
| IRQ           | (Not used) |
| GND           | GND        |
| RST           | 9          |
| 3.3V          | 3.3 V      |
